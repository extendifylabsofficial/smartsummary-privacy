üìäData Collection

üö´We Do NOT Collect Any Data

SmartSummary does not collect, store, transmit, or have access to any of the following:

  • Personal Information: No names, emails, or contact details
  • Browsing History: No tracking of websites you visit
  • Webpage Content: No storage of pages you summarize

  • Cookies or Tracking: No cookies, pixels, or tracking mechanisms

Technical Implementation

SmartSummary is designed to operate completely offline after installation. The extension uses only:

  • chrome.storage.local - For saving your preferences and summaries locally
  • chrome.tabs - To access the current page content for summarization
  • chrome.scripting - To inject content analysis scripts

No network permissions are requested or used.

Local Processing Flow

1
You visit a webpage and click "Generate Summary"
2
SmartSummary extracts text content directly from the page's DOM
3
JavaScript algorithms analyze the content locally on your device
4
A summary is generated and displayed in the extension popup
5

Important: At no point in this process is any data transmitted to external servers or third parties.

üíæData Storage

Local Browser Storage Only

SmartSummary stores the following data locally in your browser using chrome.storage.local:

  • Saved Summaries: Text summaries you choose to save (stored locally)
  • User Preferences: Theme settings (dark mode, high contrast)
  • Extension Settings: Summary length preferences, accessibility options
  • Pro Mode Status: Demo subscription activation state

Storage Details

  • Location: Your browser's local storage (never synchronized to cloud)
  • Access: Only the SmartSummary extension can access this data
  • Retention: Data persists until you uninstall the extension or clear browser data
  • Size Limit: Chrome's 5MB local storage quota applies
  • Encryption: Data is stored using Chrome's standard local storage security

Data Control

You have complete control over your data:

  • Clear Summaries: Use the "Clear" button to delete all saved summaries
  • Reset Settings: Uninstall and reinstall to reset all preferences
  • Export Data: Copy summaries to external applications as needed
